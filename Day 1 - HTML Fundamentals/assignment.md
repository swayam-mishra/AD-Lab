# Day 1 - HTML Fundamentals

## Part 1: Introduction to HTML

**Assignment:** Create your résumé as a webpage.

### Requirements
- Use structural tags: `<html>`, `<head>`, `<title>`, `<body>`
- Include heading tags (`<h1>` through `<h3>`) for name, section titles
- Use paragraphs (`<p>`), line breaks (`<br>`), and horizontal rules (`<hr>`)
- Add an ordered or unordered list for skills/achievements
- Include at least one hyperlink (`<a>`) and one image (`<img>`)
- Apply text formatting (`<b>`, `<i>`, `<strong>`) where appropriate

Reference implementation: `1. Introduction to HTML/index.html`

## Part 2: HTML Form Design

**Assignment:** Create a **Campus Placement Registration Form** using HTML.

### Requirements
- Use a `<form>` element with appropriate `action` and `method` attributes
- Include the following input fields:
  - Text inputs: Full Name, Email, College, Branch
  - Password field: Create a PIN
  - Radio buttons: Gender selection
  - Checkboxes: Skills / Technologies known
  - Dropdown (`<select>`): Year of Graduation
  - `<textarea>`: Brief introduction / About yourself
  - Submit and Reset buttons
- Wrap related fields in `<fieldset>` with `<legend>` labels
- Use `<label>` elements linked to each input via `for` attribute

Reference implementation: `2. HTML Form Design/index.html`
